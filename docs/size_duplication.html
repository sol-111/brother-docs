1サイズに複数デザインの可能性

同じサイズのデザインを複数登録できる必要があるかご確認ください

質問 1つのテンプレート内で、同じサイズのデザインを複数持つことはありますか？
お聞きしたいこと

1つのデザインテンプレート内で、たとえばA3サイズのデザインを2種類以上登録したいケースはありますか？
それとも、各サイズにつき1種類で十分でしょうか？

📋 具体的にはこういうことです

現在は「1つのデザインテンプレートに対して、各サイズ（A4・A3・長尺タテ・長尺ヨコ）のデザインは1つずつ」という前提で開発を進めています。

「春のデザイン」テンプレートの場合

現在の想定：各サイズ1種類
A4 → デザイン 1つ
A3 → デザイン 1つ
長尺タテ → デザイン 1つ
長尺ヨコ → デザイン 1つ
もしサイズ重複ありの場合
A4 → デザイン 1つ
A3 → デザイン 2つ
長尺タテ → デザイン 1つ
長尺ヨコ → デザイン 3つ
パターン A

各サイズ1種類で十分

1つのデザインテンプレート内で、同じサイズのデザインは1つだけ。シンプルな構成で管理しやすい

パターン B

同じサイズで複数必要

将来的にA3で2パターン等の可能性があるため、同じサイズに複数デザインを登録できるようにする

💡 現在の開発方針：パターンA（各サイズ1種類）で進めています。パターンBの場合はアップロード画面のUI設計が変わるため、お早めにご連絡いただけると助かります。
